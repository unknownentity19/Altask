<Hero/> Component

The Hero component renders a headline, subheading, and call-to-action area. Used as the first block of most marketing pages.

PropTypeDefaultDescription
titlestring—Headline text
subtitlestring—Supporting copy
cta{ label, href }—Primary call-to-action
variant"centered" | "split""centered"Layout style
theme"light" | "dark""light"Color theme
// install npm install @altask/components // usage import { Hero } from "@altask/components"; export default function Page() { return ( <Hero title="Build websites visually" subtitle="No code. Full control." cta={{ label: "Start free", href: "/editor" }} variant="centered" /> ); }

POST /v1/sites

Create a new site programmatically. Returns the site ID and a publish token.

# curl curl -X POST https://api.altask.app/v1/sites \ -H "Authorization: Bearer $ALTASK_KEY" \ -H "Content-Type: application/json" \ -d '{"name": "habits-app", "template": "lift"}' # response { "id": "site_8aL2k9X", "url": "https://habits-app.altask.app", "publish_token": "pt_••••••••" }
