Drag & Drop Builder

Compose pages by dragging,
never by typing CSS

A flexible canvas with smart snapping, alignment guides, keyboard shortcuts, and full undo history. Built for the speed of real product teams.

Try the canvas free Read the docs
What you get

A canvas built like real design tools

Smart defaults, real keyboard shortcuts, and live multiplayer — without the steep learning curve.

Smart snapping

Components align to a flexible grid as you drag. Spacing guides appear automatically. Drop-zones highlight live so you always know where a block will land.

Drag to reorder

Move blocks up, down, or into nested containers with a single drag. Or pick a block and use arrow keys for pixel-precise reordering with the keyboard.

Full undo history

80 levels of undo with redo. Every property change, every reorder, every duplicate — captured. Cmd+Z and Cmd+Shift+Z navigate any change.

Inline editing

Click any text to edit it

No popup forms. No "edit modes". Click headlines, buttons, paragraphs anywhere on the canvas and start typing. Properties update live in the inspector — and the inspector updates live as you type.

  • Inline rich text formatting
  • Markdown shortcuts (## for h2, ** for bold)
  • Auto-save on every keystroke
Keyboard-first

Shortcuts for everything

Selection

  • Select parent block Esc
  • Multi-select ⌘ + Click
  • Select all in section ⌘A

Editing

  • Undo ⌘Z
  • Redo ⌘⇧Z
  • Duplicate block ⌘D

Navigation

  • Move block up ⌘↑
  • Move block down ⌘↓
  • Toggle inspector ⌘I
