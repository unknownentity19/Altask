FAQ

Common questions

Is coding required?
No. Altask is designed for visual building. You can ship an entire site without writing a single line of code, while keeping the door open for custom code when you want it.
Can I export my website?
Yes. The free editor on this site even exports your work as a clean, dependency-free .html file you can host anywhere.
Is hosting included?
Every plan includes managed hosting on a global edge network with automatic HTTPS. Pro and Team plans add custom domains.
Is it free?
Altask has a generous free tier with the full visual builder. Upgrade only when you need a custom domain or advanced features.
