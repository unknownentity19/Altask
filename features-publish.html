Infrastructure

Production hosting, included on every plan

No bills from Cloudflare. No Vercel quota emails. No SSL renewal Slack threads.

HTTPS by default

Auto-renewing TLS certificates from Let's Encrypt. HTTP/2 and HTTP/3 served everywhere. HSTS preload included. You will never see an "insecure" warning.

Global edge network

220+ edge locations on 6 continents. Static assets served from the closest POP. Median TTFB under 120ms anywhere on Earth, including remote regions.

Auto asset optimization

Images converted to WebP/AVIF. CSS and JS minified and Brotli-compressed. Critical CSS inlined. Fonts subset. All transparent — no config to write.

Custom domains in 2 minutes

Apex domains, subdomains, and wildcard subdomains. Add a CNAME, hit verify, done. We handle DNSSEC, CAA records, and edge propagation automatically.
